## Eleventy Migration

This site is currently being migrated to Eleventy.

### Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npx @11ty/eleventy --serve
   ```

The site will be available at `http://localhost:8080`.
